# Product Overview

Tech Decision Comparator is a frontend-based decision support tool that helps users compare technology options based on real-world constraints.

## Core Purpose
- Compare technology choices (AWS services, databases, etc.)
- Provide constraint-based recommendations (Cost, Scalability, Ease of Use, Performance)
- Explain trade-offs through visual pros/cons cards

## Target Use Case
Built for the AI for Bharat Hackathon, this tool helps developers and architects make informed technology decisions by weighing different options against their primary constraints.

## Key Features
- Simple dropdown-based selection interface
- Visual comparison cards showing relevant trade-offs
- Recommendation engine that suggests the better option
- Clean, professional UI optimized for decision-making workflows
