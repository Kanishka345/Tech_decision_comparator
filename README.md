
# Tech Decision Comparator

A frontend-based decision support tool that compares technology options
based on user-defined constraints and explains trade-offs.

## Key Features
- Constraint-based comparison
- Visual pros/cons cards
- Recommendation engine
- Clean professional UI

## Tech Stack
- HTML5
- CSS3
- JavaScript

## Kiro Integration
Kiro was used to:
- Generate UI structure
- Design comparison logic
- Accelerate development with minimal manual coding
